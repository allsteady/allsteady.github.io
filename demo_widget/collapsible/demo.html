Deprecate style



1. Collapsible Demo

- This demo shows that negate css rules are able to affect the widget.

2. Collapsible-Set Demo (Basic)

- This demo shows that negate css rules are able to affect sub widgets.

3. Collapsible-Set Demo (Ignore inheritances )

- This demo shows that new defined css rules are able to ignore a inheritance style.

- 'ui-corner-all-force' : This class can ignore 'ui-corner-none' that it is inherited from collapsibleset.

- 'ui-collapsible-inset-force' : This class can ignore 'ui-collapsible-inset-none' that it is inherited from collapsibleset.
